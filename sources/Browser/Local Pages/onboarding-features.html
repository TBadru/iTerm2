What Can It Do?

Power features and thoughtful limitations

✨ Why You’ll Love It

✓
SSH Integration Magic: View files on remote servers in the browser when using SSH integration.
✓
iTerm2 Superpowers: Hotkey windows, split panes, and custom key bindings but for the web.
✓
Built for Engineers: Copy on select, broadcast input, advanced paste, and regex search simplify workflows.
✓
Productivity Tools: Instant replay, global search, named marks, and the Composer bring iTerm2 features into the browser.
✓
Browser Basics: Integrates with the iTerm2 password manager, Webkit adblock lists, and iTerm2’s AI chat feature.

⚠️ Know the Limits

!
Beta Alert: This is the first release! Expect some turbulence. As always, file an issue if something goes wrong.
!
Secondary Browser: iTerm2 doesn’t expect to be your primary browser. The purpose is to complement the terminal and facilitate particular tasks that other browser can’t.
!
Platform Constraints: Some features like Passkeys and advanced ad blocking are unavailable due to Apple’s restrictions on web views.
